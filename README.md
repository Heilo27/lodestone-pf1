# Lodestone

Pathfinder reference library apps for iOS/iPadOS.

## Apps
- **Lodestone PF1** — Pathfinder 1E (launches first)
- **Lodestone PF2** — Pathfinder 2E
- **Lodestone SF1** — Starfinder Core

## Monetization
- Free: Core rulebook
- $2/mo subscription: Expansions + accessories

## Data Source
Archives of Nethys (aonprd.com) — cross-linked references
