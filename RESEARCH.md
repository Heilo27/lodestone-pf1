# Lodestone Research — Pathfinder 1E Data Structure

## Source: Archives of Nethys (aonprd.com)

### Content Organization

**Core Sections:**
1. **Character Creation**
   - Classes (39 playable + variants, 6 companion types, 5 NPC classes)
   - Races
   - Traits
   - Feats
   - Skills
   - Prestige Classes
   - Mythic Index

2. **Game Systems**
   - Deities
   - Equipment (Non-Magical)
   - Magic Items
   - Technology
   - Spells/Rituals
   - Afflictions

3. **Rules & Reference**
   - Core Rules
   - Hazards
   - NPC Index
   - Monsters
   - Tools
   - FAQ

4. **GM Tools**
   - GM Screen
   - Useful rules in one place

### Books & Expansion Model

**Categories:**
- Core Rulebook (free tier)
- Adventure Paths
- Campaign Settings
- Modules
- Player Companions
- Supplements & Accessories

### Cross-Linking Strategy

- Every concept (class, feat, spell, monster) links to related entries
- Traits link to classes that use them
- Spells link to classes that can cast them
- Magic items link to creation rules and relevant monsters
- Deep hyperlink structure native to source material

### First Free App: Lodestone PF1

**Free Content:** Core Rulebook (classes, races, core feats, core spells, base rules)
**Subscription (\$2/mo):** All expansions, supplements, accessories, advanced options

### Data Acquisition

- Archives of Nethys offers JSON/API export-friendly structure
- All official errata integrated
- Content updates tracked by source book
